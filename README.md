# Showz - Marketing Analytics

## 📊 Descripción
Análisis de datos para optimizar los gastos de marketing de Showz.

## 🎯 Objetivos
- Analizar comportamiento de usuarios
- Calcular LTV, CAC y ROMI
- Recomendar asignación óptima de presupuesto
